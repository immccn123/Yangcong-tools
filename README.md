## 目前可以达到的效果

- 视频作业/精测作业/课后习题 100% 准确率 + 100% 认真完成视频观看（并且超长）
- 完成假期作业

## 永远不可能达到的效果

- 修改已完成作业的结果
- 帮你应付老师的线下抽查
- 将过期作业当成按时完成
